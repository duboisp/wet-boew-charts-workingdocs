Utilisation de fonction personalisé dans vos préréglages

Contexte: Vous voulez extraire et traité une donnée pour la création de votre graphique

Déclaration de votre préréglage personalisé

<script>
var wet_boew_charts = {
	charts: {
		personlise: {
			fn: {
				"/getcellvalue": function(elem){
					var raw = $(elem).text().replace(/\s\s+/g, ' ').replace(/^\s+|\s+$/g, '').replace(/N/g, '');
					return [parseFloat(raw.match(/[\+\-0-9]+[0-9,\. ]*/)), raw.match(/[^\+\-\.\, 0-9]+[^\-\+0-9]*/)];
				}
			}
		}
	}
}
</script>

La cleft à l'intéreur de l'objet fn est similaire a un pointeur JSON tel que défini dans RFC 6901. La cleft représent le chemin vers où vous désirer remplacer la valeur par la fonction définie.
